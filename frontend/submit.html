Mape: A B C

Program se šalje kao hex string, gde prva četiri karaktera predstavljaju prvu instrukciju, sledeća četiri drugu, i tako dalje. Razmaci i novi redovi su dozvoljeni i ignorišu se.

Na primer, da bi poslali program MOV R0, 0; SYSCALL R0, 5, možete uneti string 0800 d805.

Možete poslati specijalizovane programe za različite mape (tj. odabrati za koje mape će se dati program koristiti).

U polje "token" unesite tajni string koji identifikuje vašeg robota.
